h2. Contacts

* For any support on the ScattPy please send e-mail to "scattpy@googlegroups.com":scattpy@googlegroups.com
* The archive of the support conversation is stored at the "google groups page":http://groups.google.com/group/scattpy
* If you have found a defect in the ScattPy, please submit it to the project's defect tracking system at "http://github.com/ScattPy/scikits.scattpy/issues":http://github.com/ScattPy/scikits.scattpy/issues
* ScattPy git repository is hosted at "GitHub":http://github.com/ScattPy/
