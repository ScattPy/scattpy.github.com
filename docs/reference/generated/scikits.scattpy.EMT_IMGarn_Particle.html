scikits.scattpy.EMT_IMGarn_Particle¶

class scikits.scattpy.EMT_IMGarn_Particle(shape0, params, ms, volumes)¶

Particle using inverse Maxwell-Garnett effective medium theory, an approximation of inhomogeneous particles with homogeneous one having averaged complex refrecive index m. Inverse Maxwell-Garnett mixing rule is used.
